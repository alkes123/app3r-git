¿Que es una aplicacion movil?

Una aplicación móvil es una aplicación informática diseñada para ser ejecutada en teléfonos inteligentes, tabletas y otros dispositivos móviles y que permite al usuario efectuar una tarea que el programador de la app le haya asignado.

¿Que son las apps?

Clasificacion de apps de acuerdo su funcionlidad

Existen diversos tipos de apps que a continuacion te mostrare

-Básicas

Este tipo de aplicaciones se basan en su pantalla de inicio ya que en cuanto se abre,mostrara temas y enlaces a acceder, en el cual al darle click, te llevará a otro un conjunto de nuevas listas.


Juegos

Este tipo de apps, es muy amplia ya que puede abarcar juegos pixeleados, con algunos botones y con poco contenido hasta llegar a lo 3D.


Servicios personalizados

Estas aplicaciones están orientadas a permitir la entrada de los usuarios al contenido de una manera específica y la mayoría de las veces piden un nombre de usuario y contraseña o alguna información específica para validarle el acceso al usuario.

Las siglas html significa Hypertext Markup Lenguaje, donde html 5 es una mejor version que la 4ta version, ya que este ofrece una amplia extension de herramientas

¿Que es Dreamweaver?

Adobe Dreamweaver es una aplicación en programa de estudio (basada en la forma de estudio de Adobe Flash) que está destinada a la construcción, diseño y edición de sitios, vídeos y aplicaciones Web basados en estándares.

¿Que es CSS?

Las siglas CSS3 significan Cascading Style Sheet, las hojas de estilo en cascada nos ofrecen las probabilidad de definir reglas y estilos de presentación en diferentes dispositivos ya sea en PC, portátiles, móviles, etc. Las hojas de estilo nos permiten definir de manera eficiente temas, estilos y animaciones para la página o aplicación.

Es un lenguaje de programación que se usa para crear programas que son insertados en una pagina web, con javasript se pueden crear efectos e interactuar con los usuarios.

¿Que es una APK?

APK significa: Android Aplication Package, es un archivo ejecutable para sistemas operativos para dispositivos Android, este formato es una variante del formato JAR de Java y se usa para distribuir e instalar componentes empaquetados para la plataforma Android, usualmente las APKS son juegos o aplicaciones sin necesidad de usar la Play Store
